# The Cognitive Investor 📈

> How behavioral biases shaped the 2020–2023 market cycles.

## Overview
This project analyzes the price evolution of four major assets — AAPL, QQQ, SPY, and TSLA — across one of the most psychologically turbulent periods in modern financial history.

## Key Insights
- **2020 Crash:** COVID panic caused synchronized selling regardless of fundamentals
- **2021 Euphoria:** TSLA gained 700%+ driven by narrative, not earnings
- **2022 Reset:** Fed rate hikes triggered anchoring bias — investors held expecting a return to highs

## Stack
- Python (yfinance, pandas) — data extraction
- Excel — data cleaning, formatting, return calculations
- Tableau — interactive dashboard
